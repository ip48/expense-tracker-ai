## Quick Testing Guide

To fully test all features:

1. **Add expenses** from different categories over the past few months
2. **View Dashboard** to see summary cards, category breakdown, and charts
3. **Filter expenses** by category, date range, and search terms
4. **Edit an expense** to verify the update functionality
5. **Delete an expense** to test the confirmation flow
6. **Export to CSV** to verify the export feature works

## Recommended Test Data Distribution

For best visual results:
- Add 15-20 total expenses
- Spread them across 2-3 months
- Include at least 2-3 expenses in each category
- Vary the amounts between $10 and $200
- Use realistic descriptions

This will ensure all dashboard visualizations and charts display properly.
